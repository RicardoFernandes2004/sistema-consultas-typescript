## Incializar projeto  
Na raiz do projeto, abra o terminal e rode os seguintes comandos:
```bash
tsc
node dist/index.js
```
> todos os arquivos .js serão armazenados em dist

## Funcionalidades Ativas
- Criar consultas
- Confirmar consultas
- Cancelar consultas (exceto realizadas)
- Listar por status
- Listar consultas futuras
- Calcular faturamento
- Exibir dados formatados (datas e valores em português)
